# Social Network Mockup

A complete social network interface mockup built with Tailwind CSS and plain HTML.

## Features

### Main Feed (index.html)
- **Navigation Bar**: Logo, search bar, notification icons, profile dropdown
- **Left Sidebar**: User profile summary, quick links (Events, Groups, Saved Posts, Settings)
- **Main Feed**:
  - Create post interface with photo/video/feeling options
  - Multiple post examples with likes, comments, shares
  - Responsive design for different screen sizes
- **Right Sidebar**: Trending topics, suggested friends

### Profile Page (profile.html)
- **Cover Photo**: Large banner image with edit option
- **Profile Header**: Profile picture, user info, stats (posts, followers, following)
- **Action Buttons**: Follow, Message, More options
- **Left Sidebar**: About section, photo gallery, friends grid
- **Profile Navigation**: Posts, About, Friends, Photos, Videos tabs
- **User Posts**: Personal posts with engagement metrics

## Design Features

- **Responsive Layout**: Works on desktop, tablet, and mobile
- **Modern UI**: Clean, professional design with proper spacing
- **Interactive Elements**: Hover effects, buttons, and navigation
- **Real Images**: Uses Unsplash for realistic profile pictures and content
- **Icon Integration**: Font Awesome icons throughout the interface
- **Color Scheme**: Blue primary color with gray accents

## Technologies Used

- **HTML5**: Semantic markup structure
- **Tailwind CSS**: Utility-first CSS framework via CDN
- **Font Awesome**: Icon library for UI elements
- **Unsplash**: High-quality stock photos for realistic content

## File Structure

```
mockup/
├── index.html      # Main social feed page
├── profile.html    # User profile page
└── README.md       # This documentation
```

## Usage

1. Open `index.html` in your browser to view the main social feed
2. Click on profile elements or navigate to `profile.html` to view the profile page
3. All styling is handled by Tailwind CSS loaded from CDN
4. Images are loaded from Unsplash for demonstration purposes

## Responsive Breakpoints

- **Mobile**: < 640px - Single column layout
- **Tablet**: 640px - 1024px - Adjusted sidebar visibility
- **Desktop**: > 1024px - Full three-column layout

The mockup demonstrates a modern social network interface with all the essential features users expect from platforms like Facebook, Twitter, or LinkedIn.
